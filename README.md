## Info

-  **OS:** Manjaro
-  **WM:** i3
-  **Font:** Noto Sans / Noto Sans Mono

-  **Bar/Panel:** Polybar
-  **Text Editor:** Vim
-  **GTK Theme:** Coffee
-  **Application Launcher:** rofi
